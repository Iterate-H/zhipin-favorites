# Boss直聘 岗位导出工具

> 油猴脚本（Userscript），支持将 Boss直聘 **收藏岗位** 和 **搜索结果** 批量导出为 Excel + JSON 文件。

---

## ⚠️ 使用须知

- 本脚本**仅供个人学习与求职使用**，禁止用于任何商业目的
- 本脚本仅读取**你自己可访问的页面数据**（收藏夹、搜索结果），不爬取他人隐私信息
- 使用前请自行评估是否符合 Boss直聘《用户协议》及相关平台规定，**违规使用导致账号封禁等后果由使用者自行承担**
- 请勿高频使用，脚本内置随机延迟（约 1.5~2.5 秒/岗位）以降低对平台的影响

---

## 功能特性

- 📋 **收藏岗位导出**：一键导出收藏夹所有岗位完整详情，自动翻页
- 🔍 **搜索结果导出**：对任意关键词搜索结果批量导出，自动翻页
- 📊 **双格式输出**：同时生成 `.xlsx`（岗位列表 + JD 详情两个 Sheet）和 `.json`
- 🔄 **中断续抓**：进度保存在本地，随时中断，下次打开页面可一键恢复
- 🚀 **详情数据完整**：通过 XHR 拦截获取薪资、经验、学历、公司规模、HR 信息、岗位描述等完整字段
- 🧭 **SPA 感知**：自动检测页面路由切换，无需刷新即可切换导出模式

## 使用方法

### 导出收藏岗位

1. 打开 [Boss直聘收藏页](https://www.zhipin.com/web/geek/recommend?tab=4&sub=1)
2. 页面右下角出现「📋 收藏岗位导出」工具卡片
3. 点击「**开始导出**」→ 脚本自动扫描所有收藏并逐一抓取详情
4. 完成后点「**下载文件**」

### 导出搜索结果

1. 在 Boss直聘搜索框输入关键词并搜索
2. 页面右下角出现「🔍 搜索结果导出」工具卡片
3. 点击「**开始导出**」→ 脚本自动翻页收集并逐一抓取详情
4. 完成后点「**下载文件**」

### 中断与恢复

| 操作 | 说明 |
|------|------|
| 关闭页面 / 刷新 | 进度自动保存，不会丢失 |
| 继续抓取 | 下次打开页面点击「继续抓取」恢复进度 |
| 导出已抓取数据 | 中途可下载已完成部分 |
| 清除 / 重来 | 清空进度，重新开始 |

## 工作原理

```
列表页（收藏 / 搜索结果）
  └── 扫描所有岗位链接（自动翻页）
        └── 依次跳转详情页
              ├── 拦截 XHR /wapi/zpgeek/job/detail.json → 获取完整数据
              └── DOM 兜底提取（XHR 超时时使用）
                    └── 返回列表页，触发下载
```

脚本在 `document-start` 阶段注入 XHR/Fetch 拦截器，在页面最早阶段捕获详情接口响应，所有数据仅保存在本地浏览器 IndexedDB 中，不对外传输。
